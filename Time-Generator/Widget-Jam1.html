Link OBS Siap Digunakan!
 Cara Pakai di OBS
1

Klik "Copy Link" di atas

Link widget sudah dikodekan dengan semua setting yang kamu pilih. Satu link = satu widget.

2

Buka OBS → Sources → + → Browser

Beri nama source (misal: Widget Jam), klik OK.

3

Paste link di kolom URL

Di OBS, di field URL, paste link yang sudah disalin. Pastikan Local File tidak dicentang jika menggunakan path file:///.

4

Atur Width & Height lalu OK

Rekomendasi: Width 600 px, Height 200 px untuk Standard. Sesuaikan dengan kebutuhan layout stream kamu.

Link berhasil disalin!
Cara Menggunakan Widget di OBS

Ikuti langkah berikut untuk menambahkan widget ke stream kamu.

1

Atur Widget di Builder

Pilih layout, warna, font, dan format waktu sesuai keinginan. Preview akan update otomatis secara realtime.

2

Klik "Generate OBS Link"

Sistem akan membuat link unik yang menyimpan semua setting kamu. Klik Copy Link.

3

Di OBS: Sources → + → Browser Source

Beri nama, lalu di field URL paste link yang sudah disalin. Atur ukuran dan klik OK.

4

Widget langsung aktif!

Jam akan berjalan real-time di stream kamu. Jika ingin mengubah tampilan, cukup generate link baru dan update URL di OBS.
