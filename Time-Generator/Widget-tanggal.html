Cara Menggunakan Widget di OBS

      
      

        

          1
          

            
Salin Kode HTML

            
Setelah mengatur widget sesuai keinginan, klik tombol "Tampilkan Kode HTML" lalu tekan "Salin ke Clipboard".

            

              
Simpan kode tersebut sebagai file widget.html di komputer Anda.

            

          

        

        
        

          2
          

            
Tambahkan Source Browser

            
Di OBS, di panel Sources, klik tombol "+" dan pilih "Browser".

            

              

            

          

        

        
        

          3
          

            
Konfigurasi Browser Source

            
Beri nama untuk source Anda (misal: "Widget Waktu") lalu klik OK.

            
Di jendela properties:

            
    
              
  • Centang opsi "Local file"
    • 
              
  • Klik "Browse" dan pilih file HTML yang Anda simpan
    • 
              
  • Atur ukuran sesuai kebutuhan (misal: Width: 800, Height: 300)
    • 
            

            

              
Pastikan opsi "Shutdown source when not visible" tidak dicentang agar widget tetap berjalan.

            

          

        

        
        

          4
          

            
Atur Posisi Widget

            
Di preview OBS, drag widget ke posisi yang diinginkan.

            
You can adjust the size by pulling the corners of the widget.

Tentang Aplikasi

      
      

        

          

            
          

          

            Aplikasi Editor Widget Tanggal ini memungkinkan Anda membuat widget Tanggal dan Jam yang dapat disesuaikan untuk digunakan di siaran langsung OBS, streaming, atau situs web Anda.
          

          

            Dibuat dengan menggunakan HTML, CSS, dan JavaScript. Widget yang dihasilkan menggunakan Tailwind CSS untuk styling dan animasi.
          

          

            Versi: 1.0.0
